About Us

Welcome to our project! This application is a clone of the popular news-sharing platform Medium. It has been developed for educational purposes and to demonstrate modern web application development techniques.

Our Mission

Our goal is to provide a platform where users can explore, share, and publish content just like on Medium. This project showcases advanced features of Angular, including state management, component-based architecture, and material design.

Technology Stack

We are using the following technologies to build this application:

  • Angular: For the front-end framework and component-based architecture.
  • NgRx (Redux): For state management.
  • RxJS: For handling asynchronous data streams.
  • Material CSS: For styling and pre-built UI components.
  • TypeScript: For type safety and scalability.
